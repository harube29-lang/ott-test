① 콘텐츠 카드 — 세로 비율 → 와이드 비율 개선
개선 전 — 세로형(2:3) 카드, 이미지 식별 어려움
개선 후 — 와이드(16:10) 카드, 선명한 이미지
② Editor's Pick 배너 — 흐리고 어두운 이미지 → 선명한 실제 이미지
개선 전 — opacity 0.5, 이미지 구도 훼손
개선 후 — 선명도 100%, 부드러운 그라데이션 블렌딩
③ 추천 슬라이더 — 우측 대형 빈 공간 → 꽉 찬 Partial Peek
개선 전 — max-width 캡으로 우측 빈 여백 발생
개선 후 — max-width 제거, 컨테이너 꽉 채움
④ 히어로 배너 — Supabase 포스터 → 서울의밤 실제 이미지 교체
개선 전 — 작은 포스터 이미지, brightness 55% 어둡게 처리
개선 후 — 고화질 이미지, blur 제거, 좌/하단 그라데이션만
